A Travel Companion Android application built for SIT708 that helps international travellers convert essential values including Currency, Fuel Efficiency, and Temperature.

## 📱 Features

- 💰 **Currency Conversion** — Convert between USD, AUD, EUR, JPY, and GBP using fixed 2026 rates
- ⛽ **Fuel & Distance Conversion** — Convert MPG, km/L, Gallons, Liters, Nautical Miles, and Kilometers
- 🌡 **Temperature Conversion** — Convert between Celsius, Fahrenheit, and Kelvin
- ✅ **Input Validation** — Handles empty inputs, non-numeric values, identity conversions, and negative fuel values

## 🛠 Built With

- **Language:** Java
- **IDE:** Android Studio
- **Minimum SDK:** API 24 (Android 7.0)
- **Target SDK:** API 36

## 📐 Conversion Rates Used

### Currency (Fixed 2026 Rates)
- 1 USD = 1.55 AUD
- 1 USD = 0.92 EUR
- 1 USD = 148.50 JPY
- 1 USD = 0.78 GBP

### Fuel & Distance
- 1 MPG = 0.425 km/L
- 1 Gallon (US) = 3.785 Liters
- 1 Nautical Mile = 1.852 Kilometers

### Temperature
- Celsius to Fahrenheit: F = (C × 1.8) + 32
- Fahrenheit to Celsius: C = (F − 32) / 1.8
- Celsius to Kelvin: K = C + 273.15

## 🚀 How to Run

1. Clone this repository
2. Open in Android Studio
3. Run on emulator or physical device (API 24+)
